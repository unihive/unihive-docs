## Getting Started

### 1\. Clone this Repository

```
git clone https://github.com/surjithctly/neat-starter.git
```

### 2\. Navigate to the directory

```
cd neat-starter
```

### 3\. Install dependencies

```
npm install
```

### 4\. Run Eleventy

```
npm run start
```

### 5\. Build the project

```
npm run build
```
